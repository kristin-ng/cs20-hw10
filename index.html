TheMealDB

TheMealDB is an open crowd-sourced database of Receipes from around the world, offering a free JSON API for anyone wanting to use it.

TheMealDB has options for:

    Searching Meals by Name
    List all meals by first letter
    Lookup full meal details by id
    Lookup a single random meal
    List all meal categories
    List all Categories, Area, Ingredients
    Filter by main ingredient
    Filter by Category
    Filter by Area
    Filter by multi-ingredient (Patreon supporters only)
    Latest Meals (Patreon supporters only)
    Lookup a selection of 10 random meals (Patreon supporters only)
